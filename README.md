# CIS117-FinalProject: Project Gutenberg Word Frequency
Course: CIS 117 Python Final Project
Authors: Kwame Puryear and Melissa Chatelain
Date: May 12, 2026

This program runs off of the Project Gutenberg library and takes in a url from a book and scrapes what is inside to pull out the most common words. It then stores those words into a database on SQLite in order to retrieve this on demand.
